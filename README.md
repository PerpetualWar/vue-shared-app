# vue-shared-app

## Project setup
```
yarn install
```

### Compiles and hot-reloads web app for development
```
yarn serve:web
```

### Compiles and hot-reloads mobile app for development
```
yarn serve:mob
```

### Compiles and minifies web app for production
```
yarn build:web
```

### Compiles and minifies mobile app for production
```
yarn build:mob
```
